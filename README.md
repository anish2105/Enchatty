## Run Locally

Clone the project

```bash
  git clone https://github.com/anish2105/Enchatty.git
```

Create virtual environment

```bash
  python -m venv venv
  venv\Scripts\activate
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python app.py
```
